centos内核升级及docker安装

升级内核

  • 查看系统版本

  • 查看内核版本

  • 导入public key

  • 安装ml内核（主线最新版本）

  • 安装lt内核（长期支持版本）

以上两个版本二选一即可

  • 修改主配置文件

  • 重启系统即可

  • 导入public key时若报错

error: https://www.elrepo.org/RPM-GPG-KEY-elrepo.org: import read failed(2).

使用yum -y update nss即可

安装docker

  • 使用yum install docker-io安装

  • yum install docker-io，这句话是用源安装docker，可是源一般是国外源，用这个命令因为GFW被墙，根本安装不了。
